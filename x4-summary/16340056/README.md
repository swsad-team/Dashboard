​	我在这个项目中第一次接触了React这个前端库. 不谈框架语法上的东西, React对于hook的设计就让我眼前一亮, 它将组件的状态, 生命周期通过函数调用的方式抽离了出来, 能让我们更加专注于页面和业务逻辑的实现, 而且通过自定义hook的我们能进一步将相关的逻辑抽象出来, 更好的组织代码和实现逻辑的复用.
