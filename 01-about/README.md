## 1、项目简介

钱贷网是大学生通过做任务挣钱的云平台，它属于以运营为中心的服务软件，也可以理解为面向大学生的专业“众包”系统。用户可以在平台上提供悬赏发布任务，亦可以接下其他用户的任务以争取赏金。主要特点包括：

* 用户账号分为个人用户与组织用户，组织与个人均可发布自己的任务以满足不同需求；
* 用户既可以发布普通任务，通过一套确认机制完成；亦可以发布问卷调查；
